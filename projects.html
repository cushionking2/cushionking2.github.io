XHTML & CSS

XHTML and CSS are the code standards that make websites work. You can view the code behind any website by right clicking in the window and selecting "view source". Although the code you see looks a bit complicated at first, it will be easy once you understand how XHTML works.

Editors

Much like Python, websites at their most basic level are just text files. For web programming, your compiler is a web browser, so all you need in order to write a website is a simple text editor.

<Tags>

The language of XHTML revolves around tags. Every component in a website is contained within a tag. The first tag you create on a website declares the doctype, or the type of file your document will be.

Scope

Most tags have opening and closing tags. Everything in a web object is contained within the opening and closing tags.

<Head>

The head of a site includes information for the browser. The information in the head is not seen by the user of the site. It includes keywords for search engines, the title of the website, browser information etc.Below we will create a header of our own for our new website.

<Body>

The body of the site includes all of the actual site content, including paragraphs, images, audio, animations backgrounds, and so on.

Paragraphs <p>

HTML is not sensitive to white space, so spaces, tabs, and returns in your code do not translate into text on the page. The browser looks at your text as in one long line of code. <p> (and its closing tag) are used to enclose a paragraph of text on a new line.

Inline Text Tags

These tags are used to modify or display text. Line Break and Horizontal Rule do not have opening or closing tags, but ones that change text apearance do.
Line Breaks <br/>
Instead of using new <p> to separate paragraphs, <br/> can be used to create a new line and separate text.
Horizontal Rule <hr>
Similar to <br/>, it adds a horizontal line instead of a new line.
<strong>
Makes the enclosed text bold.
<em>
Makes the enclosed text italicized.

Comments

Comments in code can be added using the format: <!-- Comment Text -->

Links

Links can be added to a webpage using the <a> tag.

Headers

Header tags <h1> to <h6> are used to enclose headers in decreasing font size.

Page Navigation with Links

You can create links that go to separate parts of the page by naming your sections.

Adding Images to Your Site

Images can be used and adjusted on your site using tags and attributes. The height and width attributes can be used to adjust image size. Either pixels or page ratio can be used to specify.

Tool Tips

Tool tips allow the user to hover over elements of a page for further explanation of what they do.
